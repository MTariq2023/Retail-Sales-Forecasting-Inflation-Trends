Retail Sales Forecasting & Inflation Trends (U.S.)

Nominal vs inflation-adjusted retail sales using Census + CPI (1992–2025)

← Back to Home

Overview

This project cleans and merges U.S. Census monthly Retail & Food Services Sales with BLS/FRED CPI to produce inflation-adjusted (real) sales and visualize long-term trends in Tableau. It highlights how purchasing power changes over time even when nominal sales are rising.

View Interactive Dashboard View Code on GitHub

Tech Stack

  • Python (Pandas) for cleaning, merging, CPI indexing
  • Tableau Public for visualization (monthly timeline, nominal vs real)
  • CSV/XLSX inputs for reproducibility

Data Sources

  • U.S. Census: Retail & Food Services Sales (1992–Present) — Excel
  • FRED/BLS: CPIAUCSL (seasonally adjusted) — CSV

What I Did

  • Parsed multi-sheet Census workbook (one sheet per year), auto-detected headers, extracted “Retail and food services sales, total”.
  • Merged with CPI, indexed CPI to the latest year, computed Real_Sales.
  • Exported adjusted_retail_sales.csv for BI tools and built a Tableau line chart (Nominal vs Real).

Reproduce

  1. Clone the repo (link above) and place raw files in data/:
    • retail_sales_us_1992_2025.xlsx
    • cpi_us_monthly_1992_2025.csv
  2. Run the Jupyter notebook to generate data/adjusted_retail_sales.csv.
  3. In Tableau: connect to the CSV → Date on Columns (Month, continuous) → Nominal & Real on Rows → add Year filter.
